# Vanlife Culture Site

Drag `index.html` to Netlify/Vercel for instant deploy (vanlifeculture.vercel.app).

## Custom
- Formspree: Free account → get ID → replace `YOUR_FORM_ID`.
- Images/Vids: Swap Unsplash → your van pics (Imgur/CDN).
- Domain: Vercel/Netlify → add vanlifeculture.com.

Live preview: Open index.html in browser.
